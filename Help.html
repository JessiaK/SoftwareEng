Frequently Asked Questions :)

1) Can I try the Budget Game Application before I buy it?

         YES, we do offer users the oppurtunity to access the application for a certain time period before they buy it.

2) Can other people visit my account?

No, all user accounts will be always kept private and can only be accessed by the original user.

3)  Is it possible to set a different picture for my profile rather than the video game character?

No, unfortunately we value our users privacy thus in order to maintain  the confidentiality of each user account, it will not be possible to upload any other picture.

4) How can I delete my account?

If you are no longer using the application, you can just go to settings and click delete. This will permanently delete your account and all your information.

5) Is it possible to pick a different video game character after the sign in process?

Yes, the user will have a choice to pick their favorite video game character. The change can be done using the setting icon on the home page of the application.
